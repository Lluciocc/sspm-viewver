# SSPM Viewer

SSPM Viewer is an unofficial map viewer for Rhythia.

## Demo

You can try the demo here:  
https://sspm.lluciocc.fr/

---

## Contributing

The project is built with TypeScript and Vite.

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```
